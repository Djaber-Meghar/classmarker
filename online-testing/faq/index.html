Tests: Getting started

The Tests section is where you add and edit questions and tests. Access the Tests section either via the main navigation or via the quick-links displayed at top right of your screen.

With ClassMarker you can create as many questions and tests as you like.

Adding questions:
ClassMarker provides 3 ways to add questions to your account.
  1. Add questions to the Question Bank. All tests can select questions from the Question Bank, and each question can be re-used any number of times.
  2. Create a test, and then add questions directly into the test. These questions are also automatically added to the Question Bank which means they can be used in other tests if required.
  3. Prepare batches of questions offline and import them to ClassMarker via the Question Bank.


Adding tests:
When you create a test you will find two options for selecting the questions that will be displayed when your test is taken.

1. Adding Fixed questions means the same set of questions is displayed each time the test is taken.
2. Adding Auto-Selected questions means questions are automatically selected from specified categories within your Question Bank. Each time the test is taken, a different set of questions is displayed.
You can combine these two methods if you wish and your test will display a mix of fixed and auto-selected questions.

Once you have created your questions and tests you'll want to provide access to them. ClassMarker refers to this as "Assigning" your test; look out for the Assign buttons across the site.

Setting

When you assign (provide access to) a test (either by Group or Link) you will be provided with a number of customizable settings to ensure your test is delivered according to your requirements. These include time & date availability, number of attempts allowed, whether a certificate is to be included, the pass mark, pass and fail feedback messages and many more. Look out for the (?) symbols on the settings page for individual explanations.

Default Settings
Once you have selected your settings, you will save them at the bottom of the page. Each time you assign a test you are also given the option to save these settings as default, which means they will be preloaded for you next time you assign a test.

Questions: Getting started

ClassMarker offers the ability to create a number of different question types including: multiple choice, multiple response, true/false, short answer (defined response) and essay (longer text, response not defined).

You can add questions using any of the following methods:
  1. Add questions directly to your Question bank (go to Tests > Question bank and use the links in Manage Questions).

Why do certain foreign language characters not appear correctly?

Please check what file type are you using to edit your question template:
Windows default text editor "Notepad" will not format the question text correctly. You need to use an advanced text editor such as Notepad++ which can be downloaded for free.

If you're already using an advanced text editor, please ensure that you are not copying and pasting from Microsoft Word as this can sometimes create characters that are not UTF-8 compatible. If your characters are not UTF-8 compatible they will not format correctly.

This means you may need to source those specific characters from a different source such as a UTF-8 website, for example: https://www.utf8-chartable.de/ where you can copy and paste the required characters into your .txt template.

What are my options for multiple response question grading styles?

You can set Multiple response questions to give 'full' points only when all the correct answers and no incorrect answers are selected, or allow for partial points to be awarded for correct answers.

There are three grading methods.

Partial scoring options:
  • Off - Full points for full correct answer, zero points if an error is made
  • Partial points with deduction - Remove partial points for incorrect answers
  • Partial points - Do not remove points for incorrect answers
Example: Partial scoring with deduction
A question worth 3 points with 3 correct answers will gain 1 point for each correct answer selected but also lose 1 point for each incorrect answer selected.

Example: Partial scoring
A question worth 3 points with 3 correct answers will gain 1 point for each correct answer selected.

Partial scoring will not give negative points. Zero is the minimum points awarded.

Note: ClassMarker will limit the amount of selections allowed to the amount of correct answers given for the question to stop a user selecting all answers.

You can choose your grading style settings on a per question basis when added or editing your multiple response questions.

How do I integrate my own website and user IDs with a ClassMarker test?

ClassMarker optionally allows you to integrate your own user ids with test results allowing you to easily export this data or send data (including test results) back to your own website in real time.

You can record your own users ids by automatically appending them to the Link your users click to take their test (if they are logged in to your website for example).

Example:
https://www.classmarker.com/yourtestlink?...&cm_user_id=123

The cm_user_id will hold your users "user id for your system" and will save this against their test results.

You can also require tests can only be taken if a user_id is supplied, and limit each user id to one attempt only per test.

ClassMarker also lets you set a return link for users to follow back to your website after they have completed their test.

Return links can also have the users details, test score and duration appended to it for you to record in real time on your own website.

Example:
https://yourLINK?cm_ts=24&cm_tp=72&cm_td=0-12-35&
cm_fn=John&cm_ln=Smith&cm_e=john@example.com

Where:
cm_ts=Test Score
cm_tp=Test Percentage
cm_td=Test Duration (0-12-35 = 0h 12m 35s)
cm_fn=first name
cm_ln=last name
cm_e=email address

Return custom variables
You can also add your own unique parameters to be returned to you on the return link, by pre pending the parameter names with cm_return_

Example: The parameter cm_return_employee_id=1234 will be returned to you as &cm_return_employee_id=1234

When testing non-registered users via Link or embedding tests within your website, as well as password protecting your tests, you also have the option to limit exactly who can access your tests by using 'Access list'.

An Access list is a list of unique identifiers you add (email address, employee ids, etc.) that a user must use to access your test/s. You can limit how many times each unique identifier can be used per test and they are recorded along with the respective test results.

Both access list items and cm_user_id values can be exported to .CSV files to open in Spreadsheets with test results, date taken, test duration and IP address.

These methods do not require users to be registered to take your tests, and if using access lists you can mass import identifiers when creating and editing lists.
