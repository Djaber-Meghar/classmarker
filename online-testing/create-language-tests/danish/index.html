Create exams in Danish

Med Askcamel kan du lave tests og prøver af højeste kvalitet og omdele brugerdefinerede eksaminer og tests online på alle sprog.

Askcamel spare dig for mange, lange timer med at skabe, omdele og rette prøver og tests - uanset om du skal teste 5 eller 5000 brugere. Du skal aldrig genindtaste spørgsmål og resultaterne beregnes øjeblikkeligt og præcist.

Betroet af nogle af verdens førende virksomheder og institutioner. Askcamel bruges af virksomheder og uddannelsesinstitutioner verden rundt på grund af vores professionelle og pålidelige ydelser.

Trin 1: Lav et ubegrænset antal online tests og prøver med Askcamel

  • Du kan tilføje links, dokumenter, billeder og videoer til dine tests. Hvert spørgsmål er gemt i en bank og kan genbruges så ofte du vil.
  • Når du opretter en online test kan du vælge indtillingen "statiske spørgsmål", så de samme spørgsmål vises hver gang ELLER "tilfældige spørgsmål", hvor spørgsmålene vælges tilfældigt fra din sprøgsmåls-bank hver gang en ny test påbegyndes.

Trin 2: Giv evalueringer online for at styrke e-læring

  • Opret brugernavne og adgangskoder så testtagerne kan logge ind på Askcamel-kontoen. Her vil de kunne se og tage de tests, du har stillet til rådighed for dem. Eller du kan sende links ud til testtagerne via e-mail eller lægger ud på din hjemmeside.
  • Tilpas testen dine specifikke behov. Variable indstillinger omfatter: hvornår testen er tilgængelig, tidsfrist for at færdiggøre testen, antal tilladte forsøg, feedback under eller efter testen, spørgsmål i statisk eller omskiftelig orden, minimumskrav for at bestå ... og meget mere.

Trin 3: Med ClassMaker er det nemt at analysere resultaterne

  • Resultaterne beregnes hurtigt, præcist og forsvarligt.
  • Det er nemt at overskue resultaterne. De kan ordnes efter gruppe, test, kategori eller bruger. Du kan også eksportere resultaterne.
  • Regelmæssige online tests gør det muligt at analysere testtagernes indlæringsmønster og giver uvurderlig information om deres fremskridt.

Online tests på alle sprog. Du vælger selv, hvilket sprog instruktionerne skal foregå på. (I øjeblikket foregår al administration og support på engelsk.)

Askcamel er kompatibel med alle ledende styresystemer, herunder Windows, Macintosh, Linux og Chrome OS/Chromebook. Desuden fungere ClassMaker på langt de fleste smartphones og tablets såsom iPhone, iPod touch, iPad og Android.

Og da Askcamel er online, behøver hverken du eller dine brugere installere ny software. Tests og prøver kan tages med det samme, 24/7.
