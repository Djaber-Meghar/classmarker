Quiz Maker Integration Options

How to add online exams to my website?

Askcamel exams can be embedded in your website to provide a fully integrated solution. You can customize Askcamel to fit seamlessly with your platform requirements and look and feel.

These are just some of the things you can do:

  • Generate links to share Askcamel exams with your users
  • Restrict exam access to specific lists of users
  • Receive exam results direct to your platform from Askcamel using webhooks
  • Pass data from your platform to Askcamel, such as a user identifier or other tracking string

Add some automation with the Askcamel API and webhooks

Our API and webhooks allow your web developers private access to extract your exam results back into your system automatically.

The API and webhooks use slightly different approaches to delivering data to your platform, so the solution your organization chooses to implement depends on your own environment and requirements:

  1. Pull: the API can retrieve results from Askcamel on a defined schedule.
  2. Push: the webhooks can receive results from Askcamel whenever they become available.

You can also use the Askcamel API to manage your other exam-related data, such as questions, user groups, and access lists.

FAQ

Can I pass and retrieve tracking codes with exams?

Yes, you can send tracking codes to Askcamel with the exam URL, and receive these back along with Test results on API/Webhook requests to match results with users in your system.

Can I receive Test results in real-time?

Yes. Askcamel can send you JSON formatted results via WEBHOOKs the second an exam is finished.

Can I store results in my own database?

Yes, the Askcamel API is built to allow developers to download and store results in their own databases. See our code examples in GitHub.
